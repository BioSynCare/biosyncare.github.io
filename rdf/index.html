BioSynCare RDF

Namespaces:

  • bsc: https://biosyncare.github.io/ont# — BioSynCare core ontology namespace. Human doc · OWL/Turtle
  • id: https://biosyncare.github.io/id/ — Stable identifiers for instances (projects, sessions, protocols, outcomes).
  • sso: https://biosyncare.github.io/ont/sso# — Sensory Stimulation Ontology namespace. Human doc · TTL

Core

  • bsc-owl.ttl — Minimal OWL core defining classes (Project, Protocol, Technique, Session, Outcome, Report, ParticipantGroup) and core properties (usesTechnique, appliesTechnique, hasOutcome, partOfProject, occurredDuring, etc.).
  • bsc-skos.ttl — SKOS ConceptScheme for techniques and outcomes (e.g., NeurosensoryStimulation, BinauralBeats, MigraineRelief). Useful for lightweight tagging and UI vocabularies.
  • summary.html — Auto-generated overview of classes, subclass relations, object/datatype properties, and SKOS concepts from all TTL files.

Datasets

  • history-2025-11-05.ttl — Triplification of docs/history.md. Includes techniques, outcomes, a protocol, and example sessions with provenance (PROV) and triplification date.

External

  • sso-ontology.ttl — TTL form of the SSO vocabulary hosted here; use prefix sso: = https://biosyncare.github.io/ont/sso#.
  • sso-ontology-extended.ttl — Extended TTL with additional classes/properties used in prior work.
  • sso-initial.owl — Original SSO snapshot (RDF/XML).
  • sso-updated.owl — Updated SSO snapshot (RDF/XML).
  • onc-ontology-attachment-2.ttl — ONC Attachment 2 (TTL) aligned for reference and cross-mapping.
