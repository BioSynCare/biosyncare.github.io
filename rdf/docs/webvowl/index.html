WebVOWL integration

WebVOWL renders ontologies from VOWL-JSON files. Use the links below to open the viewer and load the JSON by URL. If the JSON files are missing, run make rdf-webvowl and push.

JSON outputs (to be generated)

  • BioSynCare core — JSON: /rdf/docs/webvowl/bsc.json
  • SSO — JSON: /rdf/docs/webvowl/sso.json

Self-hosted viewer (recommended)

We can serve the WebVOWL app directly from this site for a fully serverless setup. Install assets locally with make rdf-webvowl-viewer (copies the gh-pages branch of WebVOWL under /rdf/docs/webvowl/app/), then push.

  • Self-hosted app entry: /rdf/docs/webvowl/app/webvowl.html (will 404 until installed)
  • Launcher page: /rdf/docs/webvowl/viewer.html (prefers self-hosted; falls back to mirror)
